Community chat that belongs to the community

Existing platforms collect your data, lock features behind subscriptions, and make decisions that benefit shareholders over users. V1stax flips that model — you run it, you own it, and the AGPL license ensures no one can quietly take it private. Privacy by Default

No analytics, no telemetry, no fingerprinting. Registration requires only a Runner ID and password — email is optional, no real name, no phone number. Data minimization is architecture, not a setting.

No Paywalls. Ever.

Chat is the product. Sending messages, creating nodes, inviting people, reading history — none of it will ever be gated. The software will never introduce artificial friction to push toward a paid plan.

Self-Hosted First

A fully functional instance should run on a $5/month VPS with a single set of commands. Not a watered-down version — the full thing. No features that depend on external managed infrastructure.

Calm Interface

V1stax is infrastructure software, not a social platform competing for attention. No notification counts designed to create anxiety, no engagement nudges. The UI feels like opening a tool — not walking into a casino.

What's Shipped in v0.6.2-alpha

Real-time WebSocket messaging with typing indicators and presence, 1:1 DMs, file and image attachments, emoji reactions, markdown rendering, @mentions, thread replies, message search, pinned messages, space categories, announcement spaces, custom emoji per node, full role and moderation system with audit log, avatar/banner/profile card customization, username colors, node accent colors, mobile-first layout, and a distinct visual identity — Orbitron headers, indigo/teal palette, EQ-bar typing animation, dot-grid backgrounds. No Discord fingerprints.
