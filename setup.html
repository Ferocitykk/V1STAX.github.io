Setup

Get ATLAS V1 running locally and connected to your Twitch channel.

Prerequisites

What you'll need

Python 3.10+, a Twitch developer account with an app registered, and Ngrok for exposing your local Flask server to Twitch's webhook endpoints.

Installation

  1. Clone the ATLAS repository from the V1STAX GitHub org to your local machine.

  2. Install dependencies: pip install -r requirements.txt

  3. Copy .env.example to .env and fill in your Twitch credentials and API keys.

  4. Start Ngrok on the Flask port: ngrok http 5000 — copy the forwarding URL into your .env.

  5. Launch ATLAS: python atlas_main.py

Twitch Integration

Connecting to Chat

ATLAS connects to Twitch IRC using your bot account's OAuth token. Once running, it will join your channel and respond to commands. Event subscriptions (follows, subs, raids) are handled via Twitch EventSub through the Ngrok tunnel.
