Philosophy

Privacy by Default

No analytics, no telemetry, no fingerprinting. Registration requires only a Runner ID and password — email is optional, no real name, no phone number. Data minimization is architecture, not a setting.

No Paywalls. Ever.

Chat is the product. Sending messages, creating nodes, inviting people, reading history — none of it will ever be gated. The software will never introduce artificial friction to push toward a paid plan.

Self-Hosted First

A fully functional instance should run on a $5/month VPS with a single set of commands. Not a watered-down version — the full thing. No features that depend on external managed infrastructure.

Calm Interface

V1stax is infrastructure software, not a social platform competing for attention. No notification counts designed to create anxiety, no engagement nudges. The UI feels like opening a tool — not walking into a casino.

Community Ownership

This project doesn't belong to a founder, a company, or a VC. The AGPL license enforces this structurally — anyone who forks and hosts a modified version must also share their changes. You can't quietly take this private.

Mobile First-Class

A community chat that doesn't work on a phone isn't a community chat. Safe-area insets, no iOS auto-zoom, keyboard-aware layout, visible touch targets, swipe-panel navigation. Mobile breakage is a first-class bug.

What's Shipped — v0.9.0-alpha

Voice

WebRTC peer-to-peer voice channels built directly into nodes — no relay servers required on a LAN, STUN for NAT traversal when needed. Direct message voice calls with incoming ring tone, accept/decline modal, and automatic hang-up detection. AI noise suppression via RNNoise (WebAssembly) with per-user toggle, on by default. Individual microphone and speaker device selection, per-peer speaking indicators that light up avatars in real time, and a live call timer. Call records are inserted into DM history — completed calls show duration, missed calls are clearly labeled. A persistent call bar sits above the user panel in both sidebars so you always know you're in a call, no matter where you are in the app.

Communication

Real-time WebSocket messaging with typing indicators and presence, 1:1 direct messages with unread badges, group DMs (up to 10 runners, optional name, history from join time), friends system with friend requests and presence, file and image attachments with inline previews in both spaces and DMs, emoji reactions with live sync, markdown rendering with syntax highlighting, @mention notifications with sound and browser push, inline message replies, message edit and delete, message search within a space and within DMs, pinned messages, browser notifications for mentions and DMs with context-aware copy (group name shown in group DM notifications), notification when added to a private space, and unread message dividers.

Spaces & Community

Space categories with drag-to-reorder, spaces draggable between categories, announcement spaces, private spaces with individual member invites, private categories that hide entire sections, custom roles with granular permission controls (manage node, manage spaces, manage roles, see private spaces, kick/ban, manage messages), role color assignment with no-color option (lets runner's personal name color show through), role drag-to-reorder, per-member role assignment, two-tier base roles (Owner, Runner) with all elevated access via custom roles, member management with ban/kick, multiple invite links per node, audit log, and opt-in public node listing.

Identity & Customization

Custom avatar upload with crop UI, custom node icon and banner, user profile banner, username color picker (11 presets + custom hex), node accent color, user bio and status messages, custom emoji per node (PNG/GIF, up to 200), and profile card modals with DM button.

Instance Administration

Instance admin role with gated operator dashboard, node registry with per-node storage tracking, suspend/restore and invite controls, instance-wide stats (uptime, connections, storage, max upload size), registered user list with online status and instance-level suspension, and audit log entries for all instance-level actions. First registered user is automatically bootstrapped as instance admin. Security hardening: magic-bytes file validation, rate limiting on uploads and DM creation, automated nightly SQLite backups with 7-day rotation, JWT 1-hour expiry with proactive refresh, and account deletion with cascade or ownership transfer.

Design System

Orbitron headers throughout, indigo/teal palette — #6366f1 accent, #2dd4bf secondary, #07090d base. Square node icons with glow ring on active selection, inset left-border indicator on active space, dot-grid message background, EQ-bar typing animation, thin hairline border system, full SVG icon set. Nothing looks like Discord.

Stack

Backend

Python 3.12, FastAPI, SQLite, WebSockets. No ORM migrations framework — schema changes run as idempotent ALTER TABLE statements on startup. No external services required at runtime.

Frontend

React 19, Vite, Axios. No state management library — state lives in MainPage.jsx. Shared WebSocket base hook with exponential backoff, ping/pong keepalive, and visibility-change reconnect for mobile Safari.

Auth

JWT via python-jose, bcrypt via passlib. Rate limiting on all auth endpoints — sliding window in-memory limiter. Account recovery via encrypted recovery codes. No OAuth dependency, no email required.
