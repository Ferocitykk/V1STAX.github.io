Pi 4 Server

A personal Minecraft server running headless on a Raspberry Pi 4 — the project that started it all.

Online
Jan 25, 2025 — 1 Year Running
Friends Only

Overview

Where it started

This was the first real project — a headless Raspberry Pi 4 running Linux with no monitor, no GUI, managed entirely over SSH. Setting this up was the introduction to self-hosted infrastructure, Linux administration, and keeping a service running 24/7 on local hardware. It's been online since January 25th, 2025 and just hit its one year anniversary.

Hardware

Running

Raspberry Pi 4 — 8GB

Headless setup, no display. Managed entirely over SSH from any device on the network. Runs 24/7 as a dedicated server node.

Local Network

Home Infrastructure

No cloud hosting — fully self-managed on local hardware. Access is limited to friends with the server address.

Software

Active

PaperMC

High-performance Minecraft server software built on top of Spigot. Optimized for low-resource hardware like the Pi 4.

Active

PineCraft

Pi-optimized Minecraft distribution. Tuned specifically for Raspberry Pi hardware to keep the server stable and performant.

Minecraft 1.20

Vanilla — No Mods

Running vanilla 1.20 with no mods. Clean survival experience for a small group of friends.

Roadmap

Planned

Upgrade to a newer Minecraft version when time allows — a previous update attempt broke the server so it's been stable on 1.20 since.
