Features

A breakdown of every system inside ATLAS V1: Glitched Core.

Voice System

Custom TTS — V1 Style

ATLAS uses a modular TTS pipeline styled after the V1 voice from ULTRAKILL. Startup phrases, event triggers, and chat responses all route through this layer. Fully swappable voice backend.

Emotion Engine

Mood-Aware Responses

ATLAS tracks emotional state across a spectrum and adjusts tone, response style, and behavior in real time. Chat mood, streamer cues, and event triggers all feed into the engine.

Memory Layer

Chat + Event Memory

Short-term contextual memory for active chat sessions. ATLAS remembers recent interactions, viewer names, and key events during a stream. Vector memory integration is on the roadmap for persistent cross-session context.

Game Integration

Active

Cookie Clicker

Live game state awareness — ATLAS can read and react to in-game events.

In Progress

Minecraft

Server event hooks and chat bridging between Minecraft and Twitch.
