ATLAS V1: Glitched Core

Autonomous AI Twitch bot with voice, memory, emotion, and command logic. Powered by V1STAX.

Overview

ATLAS is your synthetic co-host — a fully autonomous AI system for stream interaction, gameplay support, lore delivery, and controlled chaos.

Core Features

  • Voice + TTS System with modular startup phrases
  • Emotion Engine reacting to chat and streamer tone
  • Memory Layer with contextual Twitch chat tracking
  • Command Routing & Flask API for modular actions

Available Commands

  • !origin – Drops ATLAS’s synthetic backstory
  • !reboot – Resets the ATLAS core for dramatic effect
  • !lockdown – Emergency containment protocol
  • !status – Outputs current emotion and system uptime
